# Reachy Mini Home Assistant Voice Assistant

A voice assistant application for **Reachy Mini robot** that integrates with Home Assistant via ESPHome protocol.

> **Note**: This is a Reachy Mini App. Install it on your Reachy Mini robot.

## Installation

Install from Reachy Mini App Store or manually:

```bash
pip install reachy-mini-ha-voice
```

## Features

- **Local Wake Word Detection** - Offline wake word detection with microWakeWord
- **ESPHome Integration** - Seamless Home Assistant connection
- **Camera Streaming** - MJPEG video stream for Home Assistant
- **Motion Control** - Head movements and antenna animations
- **30+ Entities** - Complete robot control via Home Assistant
