Installation

This app is available in the Reachy Mini Application Store. Simply click the "Install" button to add it to your Reachy Mini.

After Installation:

  1. Open the Reachy Mini Desktop App
  2. Navigate to the "Applications" tab
  3. Find "Home Assistant Voice Assistant"
  4. Click the "Run" button
  5. The app will start automatically

Configuration

The app will create a default configuration on first run. You can customize it by editing the .env file in the app directory.

Key Configuration Options:

  • AUDIO_INPUT_DEVICE - Microphone device name
  • AUDIO_OUTPUT_DEVICE - Speaker device name
  • WAKE_WORD - Default wake word (e.g., "okay_nabu")
  • ESPHOME_HOST - ESPHome server host (default: 0.0.0.0)
  • ESPHOME_PORT - ESPHome server port (default: 6053)
  • ROBOT_HOST - Reachy Mini robot host (default: localhost)

Usage

Basic Usage:

  1. Say the wake word (default: "Okay Nabu")
  2. Reachy Mini will nod to acknowledge
  3. Speak your command
  4. Reachy Mini will respond with motion and voice

Example Commands:

  • "Hello" - Reachy Mini will greet you
  • "Dance" - Reachy Mini will perform a dance
  • Any Home Assistant voice command

Troubleshooting

Common Issues:

  • Wake word not detected: Check microphone settings and volume
  • No audio output: Check speaker settings and volume
  • Cannot connect to Home Assistant: Verify network connection and ESPHome configuration
  • Motion not working: Ensure Reachy Mini is connected and not in sleep mode
